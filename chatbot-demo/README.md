# AI Chatbot Demo

Live demo of a custom AI chatbot built for businesses.

## Run locally
1. Get a free Gemini API key at aistudio.google.com
2. Open index.html in your browser
3. Paste your key and start chatting

## Features
- Gemini 2.0 Flash powered
- Custom system personality
- Clean branded UI
- One-file embed ready

## Deploy
Just host `index.html` on any static server — GitHub Pages, Netlify, or your own domain.
